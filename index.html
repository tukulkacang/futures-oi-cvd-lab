⚠️ PENTING: Pair Inverse (USD/JPY, USD/CAD, USD/CHF)

Kontrak futures 6C1/6S1/6J1 sebenarnya adalah CAD/USD, CHF/USD, JPY/USD — bukan USD/CAD dll.
Artinya price di chart naik = mata uang quote (CAD/CHF/JPY) menguat = USD melemah.

📈 BULLISH setup (price naik) → CAD/CHF/JPY MENGUAT → USD/CAD, USD/CHF, USD/JPY TURUN
   ✅ Cocok jika mau: LONG CAD/CHF/JPY atau SHORT USD

📉 BEARISH setup (price turun) → CAD/CHF/JPY MELEMAH → USD/CAD, USD/CHF, USD/JPY NAIK
   ✅ Cocok jika mau: SHORT CAD/CHF/JPY atau LONG USD

Contoh: Mau LONG JPY? → Cari setup BULLISH di 6J1 (price naik = JPY menguat = USD/JPY turun).
